Quick and dirty Bootstrap button color customization

  • Download this whole example
  • Download just bootstrap-buttons.sass

Instructions

  1. Download bootstrap-buttons.sass
  2. Replace the HEX values at the top of bootstrap-buttons.sass with your project's own kickass colors (0to255.com is a good resource, if you need to find slightly lighter or darker tints to make your buttons all look good together)
  3. Compile bootstrap-buttons.sass to a CSS file that is included BELOW Bootstrap's style sheet. (If you don't want to download the SASS gem and generally be anywhere near the command line, I recommend CodeKit for compiling SASS.)

  4. Pat yourself on the back. You've saved the web from another set of default Bootstrap buttons.
